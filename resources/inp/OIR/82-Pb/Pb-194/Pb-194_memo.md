```
Pb-194 -+-> Tl-194 --> Hg-194 --> Au-194 --> (Pt-194)
         \
          +----------> Hg-190 --> Au-190 -->  Pt-190 -+
                                                       \
                                                        +-> Os-186 -+
                                                                     \
                                                                      +-> (W-182)
```
