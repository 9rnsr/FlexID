# 体内動態モデル間のコンパートメント対応関係

```
Pb-197m -+-> Pb-197 -+-> Tl-197 --> Hg-197 --> (Au-197)
          \         /
           +-------+
```

|Kidneys1       |Kidneys1       |          |Kidneys   |

|               |RedMarrow      |RedMarrow |RedMarrow |
